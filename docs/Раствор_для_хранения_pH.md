## Изготовление 3-моль. раствора для хранения PH электродов самостоятельно.
##### Инструкция приготовления раствора для хранения PH электродов.
 **Сделаем расчеты для приготовления 3-молярного раствора KCL, для хранения и востановления PH электрода**:
# Нам понадобится [KCL](http://KCLhttps://ru.wikipedia.org/wiki/%D0%A5%D0%BB%D0%BE%D1%80%D0%B8%D0%B4_%D0%BA%D0%B0%D0%BB%D0%B8%D1%8F) желательно чда или хч (хлорид калия, он же хлористый калий)
# Молярная масса KCL = 74.5513 g/mol
# Нам нужно приготовить 3-молярный раствор, соответственно 74.5513 × 3 = 223.6539г KCL/1л. Но нам не нужен целый литр, сделаем на 100мл.
#(223.6539/1000мл) × 100мл = 22.36г KCl
Для приготовления трёхмолярного раствора KCL на 100мл, нужно 22.36 г KCL.Так что не обязательно покупать готовые растворы для хранения, можно и самому делать. Выходит то же самое, еще и дешевле. Достаточно просто купить KCL
 **Как готовить непосредственно**:

Молярная масса KCl 74,5 (39 + 35,5).
Это означает, что 1 молярный раствор будет содержать 74,5 г KCl на литр.
Следовательно, 3-молярный раствор будет содержать 74,5 x 3 = 223,5 г KCl на литр.
Теперь в 1 литре 1000 мл. Нам нужно приготовить 3М раствор в 500 мл.
Здесь просто математика. Если в 1000 мл содержится 223,5 г KCl для приготовления раствора 3M, в 500 мл будет половина этого количества (поскольку 500 - это половина от 1000).
Итак, разделим 223,5 на 2. Получим 111,75.
Итак, для начала нужно отвесить 111,75 г KCl. Для этого сначала нужно взвесить пустую чашу на весах и откалибровать весы на ноль вместе с чашей на них. Затем добавьте KCl в чашку на весах, пока шкала не покажет 111,75 г.
Затем переложите это количество в круглодонную колбу объемом 500 мл. После добавления KCl заполните бутылку наполовину и осторожно перемешайте, чтобы растворить твердое вещество. Затем доведите объем до отметки, чтобы получилось 500 мл раствора. Закройте бутылку пробкой и встряхните, чтобы перемешать. Все готово.

 Инструкция по применению от Denver Instruments для создания этого решения:

Раствор для заполнения 3M KCl можно легко приготовить с использованием хлорида калия в кристаллической
форме и деионизированной воды. Пожалуйста, внимательно следуйте инструкциям, чтобы не изменить опорный потенциал вашего электрода.

# Осторожно высушите хлорид калия в печи в течение 2–3 часов. 2) Используя аналитические весы, осторожно взвесьте 22,368 грамма KCl.
# Перенесите KCl в мерную колбу класса A на 100 мл.
# Наполните колбу до линии наполнения и тщательно перемешайте.
# Проверьте объем раствора после полного растворения KCl.
# Перелейте в оригинальную бутылку для наполнения для использования в электроде.

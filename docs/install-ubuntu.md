# Установка Ubuntu 20.04 LTS

Пример развертывания сервера на базе `ubuntu-server`.

- **Загрузить установочный образ:**
  - Обычный метод: https://releases.ubuntu.com/20.04/ubuntu-20.04.3-live-server-amd64.iso
  - Через торрент: https://releases.ubuntu.com/20.04/ubuntu-20.04.3-live-server-amd64.iso.torrent
- Нарезаем образ на флешку или CD-диск.
- При установке выбираем русский язык (важно, так как скрипты содержат русский текст).
- При выборе компонентов добавляем `OpenSSH server`, остальное установим позже через консоль `ssh`.

По завершении установки подключаемся к серверу по `ssh` (например, с помощью программы `putty` в Windows) и выполняем дальнейшую настройку.

Чтобы подключиться по `ssh`, узнаём IP-адрес командой `ip a` или `ifconfig`.

На самом сервере больше ничего руками не делаем, экран и клавиатура не нужны. Все команды будем копировать из инструкций и вставлять в консоль `ssh`.
